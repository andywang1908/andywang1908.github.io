## OAuth 2.0

OAuth 2.0 定义了三种流程：

Authorization Code Grant：适用于有服务器（换个时髦点的词吧：云服务）的第三方应用。
Implicit Grant：适用于没服务器的第三方应用，比如浏览器插件。
Resource Owner Password Credentials Grant：直接让用户输入用户名和密码授权。
